# TODAH Wedding Website 💍

## Beautiful Wedding Website for Taibat & Oluwasegun

### Features
- ✨ **TODAH Brand** - Unique trademark wedding name
- 🎨 **Teal & Onion Pink Theme** - Stunning color palette
- 💑 **Dual Events** - Traditional (June 18) & Church (July 4)
- 📸 **Guest Photo Upload** - Let guests share photos and wishes
- 💝 **Registry** - Gift registry and bank details
- ❤️ **Love Story** - Beautiful "How We Met" section
- 📍 **Complete Details** - Venues, times, dress codes
- 🎭 **World-Class Design** - Smooth animations, responsive
- 📱 **Mobile Perfect** - Beautiful on all devices

### Quick Start
```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

### What's Included
1. Hero - TODAH branding with wedding dates
2. Navigation - Smooth scrolling menu
3. Our Story - Love story with photos
4. Wedding Details - Both ceremonies with full info
5. Guest Wishes - Photo upload + messages (needs backend)
6. Registry - Gift information + bank details
7. Footer - Contact and social
