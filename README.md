## 📸 Adding Photos

### Method 1: Local Files

1. **Create folders:**
   ```
   public/
   └── images/
       ├── story/
       │   ├── couple-1.jpg
       │   ├── couple-2.jpg
       │   └── proposal.jpg
       └── venue/
           ├── traditional.jpg
           └── church.jpg
   ```

2. **Optimize images first:**
   - Visit [squoosh.app](https://squoosh.app)
   - Convert to WebP format
   - Set quality to 80-85%
   - Keep under 300KB each

3. **Update image paths:**
   ```typescript
   <img src="/images/story/couple-1.jpg" alt="Description" />
   ```

### Method 2: Cloudinary (Recommended)

1. **Sign up at [cloudinary.com](https://cloudinary.com)**

2. **Get credentials**

3. **Create `.env.local`:**
   ```bash
   NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=your_cloud_name
   NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET=your_preset
   CLOUDINARY_API_KEY=your_key
   CLOUDINARY_API_SECRET=your_secret
   ```

4. **Upload images** via Cloudinary dashboard

5. **Update image URLs** in components

## 🚢 Deployment

### Deploy to Vercel (Recommended)

1. **Push to GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: TODAH wedding website"
   git remote add origin https://github.com/yourusername/todah-wedding.git
   git push -u origin main
   ```

2. **Deploy on Vercel:**
   - Visit [vercel.com](https://vercel.com)
   - Click "Import Project"
   - Select your GitHub repository
   - Click "Deploy"

3. **Configure domain:**
   - Buy domain from Namecheap/GoDaddy
   - Add to Vercel project
   - Update DNS records

### Deploy to Netlify

1. **Push to GitHub** (same as above)

2. **Deploy on Netlify:**
   - Visit [netlify.com](https://netlify.com)
   - Click "Add new site"
   - Import from GitHub
   - Build command: `npm run build`
   - Publish directory: `.next`

## 🎭 Animations

### Framer Motion Variants

All sections use smooth entrance animations:

```typescript
// Fade in from bottom
initial={{ opacity: 0, y: 30 }}
animate={{ opacity: 1, y: 0 }}
transition={{ duration: 0.8 }}

// Slide in from left
initial={{ opacity: 0, x: -50 }}
animate={{ opacity: 1, x: 0 }}

// Scale in
initial={{ scale: 0.8 }}
animate={{ scale: 1 }}
```

### Custom Animations

**File:** `tailwind.config.js` (lines 35-60)

```javascript
animation: {
  'float': 'float 6s ease-in-out infinite',
  'pulse-slow': 'pulse 4s cubic-bezier(0.4, 0, 0.6, 1) infinite',
}
```

## 🐛 Troubleshooting

### Build Errors

```bash
# Clear cache
rm -rf .next

# Reinstall dependencies
rm -rf node_modules package-lock.json
npm install

# Rebuild
npm run build
```

### Images Not Loading

- Check file paths start with `/images/`
- Ensure images are in `public/` folder
- Clear browser cache (Ctrl+Shift+R)

### Animations Not Working

```bash
# Ensure Framer Motion is installed
npm install framer-motion
```
